## 📊 文件列表详解

### 文件标记含义
- **🔐 加密** - 该文件需要密码解压
- **✓ 无密** - 该文件无密码保护
- **💾 缓存** - 该文件的密码已成功解压过，存储在缓存中

### 快速选择
```
[✓ 全选]       - 选择所有文件
[✗ 取消选择]   - 取消所有选择
[🔐 仅选密码]  - 只选择加密的文件
```

### 手动选择
- 单击每行前的 ☐ 可选择/取消选择单个文件
- 已选择的文件会显示 ☑️

## ▶️ 开始解压

### 步骤
1. 输入要扫描的目录路径
2. 点击 🔍 扫描
3. 查看文件列表
4. 选择要解压的文件
5. (可选) 修改解压目录
6. 点击 ▶️ 开始批量解压

### 解压过程
- 系统会显示实时进度
- 每个文件的状态会实时更新
- 支持同时处理多个文件

### 完成提示
- 所有文件处理完成后，会显示"✅ 解压完成!"提示
- 点击确定后自动刷新页面

## 💡 常见操作

### 只解压加密文件
```
1. 扫描目录
2. 点击 [🔐 仅选密码]
3. 点击 [▶️ 开始批量解压]
```

### 查看成功的密码
```
1. 点击 ⚙️ 设置
2. 切换到 💾 缓存 标签页
3. 查看已缓存的密码
```

### 添加新的密码
```
1. 点击 ⚙️ 设置
2. 切换到 🔐 密码本 标签页
3. 在文本框末尾添加新密码 (每行一个)
4. 点击 💾 保存
```

### 清空所有缓存
```
1. 点击 ⚙️ 设置
2. 切换到 💾 缓存 标签页
3. 点击 🗑️ 清空缓存
4. 确认删除
```

### 切换扫描目录
```
方法一: 直接修改路径
  1. 修改顶部路径输入框
  2. 点击 🔍 扫描

方法二: 使用快捷按钮
  1. 点击 ⚙️ 设置
  2. 在 📁 目录管理 中选择目录
  3. 点击 [设置]
  4. 返回主界面重新扫描
```

## 🔔 提示

1. **首次使用**: 默认扫描 `/volume1/downloads`，可修改路径
2. **密码缓存**: 自动保存已成功的密码，无需手动操作
3. **性能优化**: 对于已缓存的文件，会自动使用缓存密码加快速度
4. **安全**: 容器重启后缓存会清空，隐私更有保障
5. **批量操作**: 支持同时选择和解压多个文件，最大化效率

## 🆘 常见问题

**Q: 为什么某些文件无法解压?**
A: 可能是密码字典中没有该文件的密码。可以在密码本中添加尝试的密码。

**Q: 如何快速解压已成功的文件?**
A: 使用缓存功能。已成功解压过的文件密码会自动缓存，下次会直接使用。

**Q: 如何修改解压目录?**
A: 修改"解压设置"区域的目录输入框，或使用默认位置。

**Q: 缓存会永久保存吗?**
A: 否。缓存保存在容器内存中，容器重启后会清空。

**Q: 如何一次解压多个目录?**
A: 需要多次扫描。先扫描一个目录、选择文件、解压，然后切换目录重复操作。
