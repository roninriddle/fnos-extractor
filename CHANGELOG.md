# 更新日志 - 功能增强 (v1.1.0)

**维护者**: Ronin

## 🎯 新增功能

### 1. 密码缓存功能 ✅
- **自动缓存**: 解压成功后自动缓存使用的密码
- **快速重用**: 下次遇到相同文件时直接使用缓存密码，无需重复尝试
- **持久化存储**: 缓存信息保存在 `/app/password_cache.json`
- **性能提升**: 对于已知密码的文件，解压速度显著加快

**API 接口**:
```bash
# 查看缓存
GET /api/password-cache

# 清空缓存
DELETE /api/password-cache
```

### 2. 子目录扫描检测 ✅
- **自动检测**: 扫描过程中自动识别包含压缩包的子目录
- **统计信息**: 显示每个子目录中压缩包的数量
- **快速导航**: 点击子目录可快速切换扫描路径
- **直观显示**: Web UI 中显示"📁 子目录检测"面板

**API 接口**:
```bash
# 扫描子目录
POST /api/subdirs
{
  "path": "/volume1/downloads"
}
```

### 3. Web 页面目录管理功能 ✅
- **快捷切换**: 一键切换到常用目录
  - `/volume1/downloads` - 默认下载目录
  - `/home` - 主目录
  - `/tmp` - 临时目录
- **自定义目录**: 支持输入任意自定义路径
- **实时更新**: 切换目录后立即更新扫描路径输入框

**位置**: Web UI 上方"⚙️ 设置"按钮 → "📁 目录管理"标签

### 4. Web 页面密码本编辑功能 ✅
- **在线编辑**: 直接在 Web 界面编辑密码词典
- **实时预览**: 显示当前密码数量
- **一键保存**: 修改后保存到服务器
- **批量操作**: 支持粘贴多行密码

**功能**:
1. 点击 Web UI 上方"⚙️ 设置"按钮
2. 切换到"🔐 密码本"标签
3. 编辑密码文本
4. 点击"💾 保存"保存更改

**API 接口**:
```bash
# 获取当前密码本
GET /api/passwords

# 更新密码本
POST /api/passwords
{
  "passwords": ["password1", "password2", ...]
}
```

## 🔧 技术细节

### 密码缓存机制
- 文件格式: JSON
- 存储位置: `/app/password_cache.json`
- 结构: `{file_path: password}`
- 自动管理: 解压时自动保存，启动时自动加载

### 缓存优先级
1. 检查缓存中是否有该文件的密码
2. 如果有,先尝试使用缓存密码
3. 缓存密码失败则尝试词典中的其他密码
4. 新的成功密码自动更新缓存

### 子目录扫描
- 扫描方式: 递归 rglob
- 识别格式: `.7z`, `.rar`, `.zip`
- 深度: 无限制
- 性能: O(n) 线性扫描

## 🚀 性能提升

### 缓存带来的性能提升
- **减少密码尝试**: 避免重复尝试密码词典
- **加快解压速度**: 缓存文件可立即开始解压
- **降低 CPU 使用**: 减少密码检测操作

**预期提升**:
- 已缓存文件: 速度提升 10-50 倍
- 平均场景: 速度提升 20-30%

## 🔐 安全注意

1. **缓存文件权限**: 缓存文件存储在容器内,容器停止后自动清理
2. **密码文本**: 在内存中处理,不记录到日志
3. **API 访问**: 建议在部署时添加身份验证层
